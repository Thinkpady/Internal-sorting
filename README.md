Лабораторная работа 3. Алгоритмы и структуры данных.

Софт, для демонстрации принципа разных видов сортировок.

Структура:
Под каждый вид сортировки реализована отдельная функция. Вывод каждого шага сортировки реализован непосредственно в ней же, за исклчением ф-ции print_mass, которая
выводит массив, не подсвечевая никаких элементов.
